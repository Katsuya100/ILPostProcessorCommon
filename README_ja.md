# ILPostProcessorCommon
## 概要
Katuusagi(a.k.a Katsuya100)の作成するILPostProcessor系ライブラリの中で使われている便利機能群です。

## インストール方法
### ILPostProcessorCommonのインストール
1. [Window > Package Manager]を開く。
2. [+ > Add package from git url...]をクリックする。
3. `https://github.com/Katsuya100/ILPostProcessorCommon.git?path=packages`と入力し[Add]をクリックする。
